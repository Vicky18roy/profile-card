# profile-card
A simple profile card made using HTML and CSS.  
This project is focused on practicing frontend design and responsive layouts.

## 🚀 Features
- Responsive profile card
- Clean UI design
- Hover effects
- Custom profile image
- Beginner friendly project

## 🛠️ Built With
- HTML5
- CSS3
